Overview

MyPasSwordX Generator is designed to operate primarily offline and does not collect personal information or store generated passwords.

Most features work without an internet connection.

Data We Do Not Collect

We do not collect, store, or transmit:

  • Names
  • Device information
  • Personally identifiable data beyond the optional recovery email
  • Generated passwords
  • User behavior or analytics
  • App usage data

All password generation and configuration settings remain stored locally on the user's device.

Activation Key (Optional)

Some advanced configuration features require a valid activation key.

When a user enters a license key, the app performs a secure online validation to check whether the key is:

  • Valid
  • Active
  • Already assigned to a device

No additional personal data is transmitted during this validation.

The key is used only for verification and authorization purposes and does not change the basic functionality of the application. Using the app without entering a key provides full functionality as described in the App Store Metadata.

Special Access Credentials

Special Access credentials are manually issued; no in-app purchase; feature works offline; general users continue full functionality.

No in-app purchase is required to use the app's core functions. The optional access credentials are provisioned externally and do not represent a purchase of additional features within the app.

Optional credentials only impact the authorized workflow module and are not required for general use.

Recovery Email (Optional)

Users may enter an email address to allow future restoration of their activation license if the device is changed or reset.

This email is stored only for:

  • License recovery
  • License ownership confirmation

The email is not used for marketing, analytics or tracking.

Entering an email is fully optional and does not affect basic functionality of the app.

No Third-Party Services

The app does not use advertising services, analytics platforms, external data processors, or tracking frameworks.

Data Security

All license-related data is stored securely and used solely for legitimate licensing purposes.

Users remain fully in control of their content and generated passwords.

Children's Privacy

The application does not collect personal information for standard use; optional license workflow may involve email.

This policy may be updated if functionality changes. The latest version will always be available on this page.
